

MDX is a combination of Markdown and JSX, and _regular_ Markdown syntax can be used for most formatting. What follows is a guide to the Markdown syntax supported by Gatsby. Each section gives the Markdown syntax, the HTML element it creates, and a demonstration of the element.

To create a line break, follow a line with two spaces and then hit enter. An alternative that is easier to see is the <br /> tag; add this tag between the two lines of text to be separated with a line break.
